# 6. FAQ

**Q1：出现“Cannot malloc bpu memory with length xxxx bytes: AllocError \{ len: xxxx\}”** 

 ![qa1](https://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/samples/zh/qa1.jpg)

A： 请确认内存模式是否切换成功

```text
# 使用 balanced 模式
/usr/hobot/bin/hb_switch_ion.sh balanced
# 重启生效
reboot
```


**Q2： 出现“ Expect HBM model in ELF format. Note that veryold HBM file not in ELF format is no longer support”** 

 ![qa2](https://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/samples/zh/qa2.jpg)

A： 请检查模型下载是否完整，可查看 md5

```text
md5sum xxxxxx

#模型 md5 值表
wget https://d-robotics-aitoolchain.oss-cn-beijing.aliyuncs.com/llm_s600/1.0.0/models/md5sum.txt
```
